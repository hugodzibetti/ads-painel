# Batch: Comprehensive App Verification

## Task 1: Type check and build
Run TypeScript type checking and build the app. Verify there are no compilation errors.
Commands:
```bash
npm run type-check
npm run build
```
Ensure both commands succeed without errors.

## Task 2: Run dev server and verify frontend loads
Start the development server and verify the frontend is accessible and loads without errors.
- Run `npm run dev`
- Wait for both server (port 3000) and frontend (port 5173) to be ready
- Access http://localhost:5173 in a browser
- Check that Dashboard, Messages, and Status pages load without console errors
- Verify API is responding (check network tab for /api calls)

## Task 3: Run bot and verify group detection
Test the WhatsApp bot functionality.
- Run `npm run bot`
- Verify QR code appears (or existing session loads)
- Check that bot lists available WhatsApp groups if group IDs aren't configured
- Verify no errors in console output

## Task 4: Docker build and run
Build and run the app via Docker to ensure production setup works.
```bash
docker compose build
docker compose up
```
Verify the app starts without errors and is accessible on port 3000 with full frontend and backend.

## Task 5: Test extraction flow
Verify the core extraction pipeline works end-to-end.
- Ensure database has test data (or messages from bot)
- Call POST /api/extract via the Dashboard "Atualizar" button or curl
- Verify activities are created and activities table is populated
- Check that no activities marked as processed=1 are reprocessed
